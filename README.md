## Client Side Docler Holding Test



### dev

    npm run dev

### build

    npm run build

### serve as static page (build first)

    cd bin
    python -m SimpleHTTPServer


### Folder structure

    - bin: Compiled files
    - config: General app configuration
    - src: Client app


### Stack

* Webpack
* Babel (Stage 0 for ES7 features)
* React.js
* Baobab (Immutable state)
* Compass
* Grunt
* Marko (Template engine)
* Socket.io
